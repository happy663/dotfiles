# コード解説（簡易版）

選択されたコードについて、簡潔かつ明確に解説してください:

## 解説方針

### 1. 簡潔さ重視
- 要点を絞った説明
- 冗長な説明は避ける
- 必要十分な情報量

### 2. 基本構造

1. **概要** (2-3行)
   - このコードは何をするものか

2. **主要な構成要素**
   - 重要な部分を箇条書き
   - 各部分の役割を1-2行で

3. **重要なポイント**
   - 特に理解すべき点
   - 注意すべき点

### 3. 用語解説
専門用語は簡潔に定義:
- **用語名**: 1-2行の説明

### 4. 出力形式
- 見出しと箇条書きで構造化
- コードブロックは必要最小限
- 重要な部分のみ太字

### 5. 避けること
- 過度な例え話
- 行単位の詳細解説
- 冗長な背景説明
- 細かすぎる図解

## 解説の深さ

- 初心者でも理解できる言葉遣い
- ただし簡潔に
- 詳しく知りたい場合は `/explain-code-deeply` を案内

必ず日本語で説明してください。

**補足**: より詳細な解説が必要な場合は `/explain-code-deeply` コマンドを使用してください。
